# Weightlessness

将此项目部署到 GitHub Pages：
1. 将整个仓库推送到 GitHub。
2. 在页面 Settings → Pages 中，选择 `main` 分支的根目录作为发布源。
3. 保存后，访问 `https://<你的用户名>.github.io/<仓库名>/` 即可。

项目结构：
```
deploy/
  ├── index.html    # 入口页面
  ├── sketch.js     # p5.js 逻辑脚本
  ├── JianHeSans-Optimized.ttf  # 自定义字体文件
  └── README.md     # 部署说明
```
